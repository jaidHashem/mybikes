# My Bikes

Simple react-native app as the requirement. Using https://mockapi.io/ for Mock data request.

## Installation

```bash
yarn
```
## Android

```bash
npx react-native run-android
```

## Android

```bash
npx pod-install
npx react-native run-ios
```
